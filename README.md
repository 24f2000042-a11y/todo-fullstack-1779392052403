# Todo App (MERN)\n\nA modern blue‑themed Todo application built with the MERN stack.\n\n## Features\n- Create, read, update (toggle complete) and delete todos\n- Responsive UI built with React + Vite\n- Axios for API communication\n- Loading states and error handling\n- Express server with CORS\n- MongoDB persistence via Mongoose\n\n## Prerequisites\n- Node.js (v18+)\n- npm or yarn\n- MongoDB instance (local or remote)\n\n## Project Structure\n
root/\n├─ backend/          # Express API\n│   ├─ server.js\n│   ├─ package.json\n│   └─ .env.example\n├─ frontend/         # React Vite app\n│   ├─ src/\n│   │   ├─ App.jsx\n│   │   ├─ App.css\n│   │   └─ main.jsx\n│   ├─ index.html\n│   └─ package.json\n└─ README.md\n\n\n## Setup\n\n### Backend\nbash\ncd backend\ncp .env.example .env   # edit if needed\nnpm install\nnpm run dev   # starts on PORT (default 5000)\n\n\n### Frontend\nbash\ncd frontend\nnpm install\n# create a .env file for Vite variables (optional)\n# e.g., VITE_API_URL=http://localhost:5000\nnpm run dev   # Vite dev server (default http://localhost:5173)\n\n\nThe frontend will proxy requests to the backend using the `VITE_API_URL` variable.\n\n## API Endpoints\n- `GET /api/todos` – list all todos\n- `POST /api/todos` – create a todo `{ title: string }`\n- `PATCH /api/todos/:id` – update fields (e.g., `{ completed: true }`)\n- `DELETE /api/todos/:id` – delete a todo\n\n## License\nMIT\n
